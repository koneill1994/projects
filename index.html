  • NLP Dashboard of COVID-19 Research Literature

    As part of the summer of Topological Data Analysis 2020, with the AFRL 711th Human Performance Wing, I and my team utilized Natural Language Processing techniques to create a dashboard that allows users to quickly visualize the existing research on coronaviruses, and search the literature against input queries and hypotheses.

    The COVID-19 Open Research Dataset is a large corpus (>200,000 documents) of articles which have been published on coronaviruses, with specific focus on research relating to COVID-19. We utilized NLP analysis techniques such as tf-idf, word and sentence embeddings such as fastText, SciBERT, and BioBERT, and dimensionality reduction techniques such as UMAP, to create an interactive dashboard which enables a user to quickly gain an understanding of the state of current coronavirus research.

    Our processing pipeline is built in python, and uses the dash framework to display data and results to the end user.

  • TDA analysis of protein secondary structure

    As part of the Summer of Topological Data Analysis 2019, with the AFRL 711th Human Performance Wing, I and my team used topological data analysis methods to predict protein secondary structure from its amino acid sequence.

    The structure of a protein determines its functions and how it interacts with other molecules, and this structure is determined by the interactions between the sequence of amino acids it is made of. Determining the structure a protein will fold into is one of the most difficult problems in biochemistry, and an efficient and accurate predictor of protein structure would be important in developing new drug treatments and biotechnology techniques.

    Our predictor is primarily programmed in python, however other components are programmed in R, or make use of the AYASDI web service implementing the mapper algorithm for topological data analysis.

  • Minimum Effort Game

    Psychological experiment software implementating the Minimum Effort Game (Van Huyck, Battalio, & Beil 1990), a game of strategic interaction which simulates situations where the payoff to each person of a group is dependent on the effort expended by the least-effortful player.

    This game is interesting in the field of game theory, and can help in investigating human responses to game-theoretical problems, such as the free-rider problem.

    This implementation supports 4 players by default, and is programmed in oTree, involving a combination of python, javascript, and html.

  • EWA model

    Software which models human behavior in the Minimum Effort Game, using the Experience Weighted Attraction (EWA) learning model (Camerer, & Hua Ho 1999)

    This implementation is programmed in R, and utilizes paralellized computation, full parameter search, and hill-climbing parameter search. I also prepared an extensive readme to help with understanding the software and my techniques.

  • FlowTetris
